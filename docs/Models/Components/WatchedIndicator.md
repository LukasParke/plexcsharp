# WatchedIndicator

Whether or not media watched indicators are enabled (little orange dot on media)

## Example Usage

```csharp
using LukeHagar.PlexAPI.SDK.Models.Components;

var value = WatchedIndicator.None;
```


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `None`             | 0                  |
| `MoviesAndTvShows` | 1                  |
| `Movies`           | 2                  |
| `TvShows`          | 3                  |
