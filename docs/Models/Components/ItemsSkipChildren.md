# ItemsSkipChildren

When found on a show item, indicates that the children (seasons) should be skipped in favor of the grandchildren (episodes). Useful for mini-series, etc.


## Supported Types

### Boolean

```csharp
ItemsSkipChildren.CreateBoolean(/* values here */);
```

### ItemsSkipChildren2

```csharp
ItemsSkipChildren.CreateItemsSkipChildren2(/* values here */);
```
