# AllowCameraUpload

Indicates if the user is allowed to upload from a camera.

## Example Usage

```csharp
using LukeHagar.PlexAPI.SDK.Models.Requests;

var value = AllowCameraUpload.Disable;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `Disable` | 0         |
| `Enable`  | 1         |
