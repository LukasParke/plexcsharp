# ChromaSubsampling

Use the specified chroma subsambling.
  - 0: 411
  - 1: 420
  - 2: 422
  - 3: 444
Defaults to 3 (444)

## Example Usage

```csharp
using LukeHagar.PlexAPI.SDK.Models.Requests;

var value = ChromaSubsampling.Zero;
```


## Values

| Name    | Value   |
| ------- | ------- |
| `Zero`  | 0       |
| `One`   | 1       |
| `Two`   | 2       |
| `Three` | 3       |
